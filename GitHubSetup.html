How to setup GitHub?

There are two parts of GitHub setup, one is on server site: http://github.com, the other is on your local PC site.

  1. Fist setup the server site:
    • go to https://github.com to sign up your github account
    • and follow the instruction on that page to change your profile.
  2. Then setup the local site:
    • from https://desktop.github.com page, click on "Dowload GitHub
      Desktop window 7 or later".
    • save the "GitHubSetup.exe" file and run the file.
    • a window pop up, click on "Install"
    • after installing, a big GitHub window popup, but you have nothing in it. If you want to clone professor's github site, there is nothing there, you have to work on the server site first, on server, you need to import professor's github site to your repository first.
  3. Import professor's github repository or create your own repository on the server:
    • login your server site github and
    • go to https://github.com/settings/repositories
    • Click on "+" on the top right corner to either create a new repository or import a existing repository.
    • To import the professor's github repository:
      • in the field "Your old repository's URL", you enter professor's site: https://github.com/dkriegls/itmd-361-spring2017
      • in the field of "You New reporsitory details:", you enter a repository name in the "Name" field (e.g., dkriegls/itmd-361-spring2017). Now you cloned the professor's repository to your repository
  4. Now on local site, you can clone any repositories on the server site to your local site:
    • double click the GitHub icon if you are not there yet.
    • on the top left corner, click "+", then click on "Clone", you will see all the repositories on the server site (including imported from professor) will be shown there, click one, and click Clone on the bottom of the screen.
  5. On server site, I can edit my file right on the screen.
  6. On local site, looks like you still need to open the Git Shell (Power shell) window and use your faverate editor to edit it.
  7. To open a Git Shell window, from the GitHub window, on the top right corner, "Open in Git Shell"
  8. to sync up the file between server site and the local site:
    • update the file from either site (server or local)
    • commit your updates
    • click the sync sign on the right top corner from the local side of screen
